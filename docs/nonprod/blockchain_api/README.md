# aid-api Non-Production Blockchain/API Toolkit

This area stores non-production validation utilities and historical sync test cases.
No runtime application code is modified by these files.

## Contents
- `scripts/nonprod/*`: helper validation scripts.
- `tests/nonprod/*/cases`: date-scoped test-case fixtures.
- `docs/nonprod/*/daily`: operator notes for each sync day.

## Usage
Run `node scripts/nonprod/blockchain_api/case_lint.js` to validate case files.
